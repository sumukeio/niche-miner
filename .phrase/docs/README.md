本目录包含 NicheMiner 项目的所有文档，按照主题分类组织。

```

│   └── 项目重构说明.md      # 重构历史记录
│
├── 02-features/             # 功能文档
│   ├── 商业意图打分功能设计.md
│   └── 长尾词筛选功能优化建议.md
│
├── 03-tools/                # 工具文档
│   ├── 百度关键词验证工具说明.md
│   └── 百度广告验证界面使用说明.md
│
├── 04-technical/            # 技术文档
│   ├── 反反爬策略说明.md

│

    └── task_mvp.md          # MVP优化任务清单
```

## 📚 文档分类说明

### 01-project/ - 项目文档
**说明**：项目级别的文档，包括项目概述、架构说明、历史记录等。

- **项目说明.md**: 项目的完整技术文档，包括功能说明、技术栈、数据库设计等
- **项目阶段性总结.md**: 项目的阶段性总结，包括已完成功能、待优化项、技术难点等
- **项目重构说明.md**: 项目重构的历史记录和变更说明

### 02-features/ - 功能文档
**说明**：具体功能的设计文档和优化建议。

- **商业意图打分功能设计.md**: 商业意图评分功能的设计方案
- **长尾词筛选功能优化建议.md**: 长尾词筛选功能的优化建议和实施方案

### 03-tools/ - 工具文档
**说明**：独立工具的使用说明和界面文档。

- **百度关键词验证工具说明.md**: 百度广告验证工具的详细说明
- **百度广告验证界面使用说明.md**: 广告验证工具的前端界面使用说明

### 04-technical/ - 技术文档
**说明**：技术实现相关的文档，包括反爬策略、配置说明等。

- **反反爬策略说明.md**: 百度反爬虫对抗策略的技术文档
- **代理IP和移动端访问说明.md**: 代理IP配置和移动端访问的技术说明

### 05-tasks/ - 任务文档
**说明**：开发任务清单和计划。

- **task_mvp.md**: MVP阶段的优化任务清单，包含15个具体任务

## 🔍 快速查找

### 按主题查找

- **想了解项目整体** → `01-project/项目说明.md`
- **想了解功能设计** → `02-features/`
- **想使用工具** → `03-tools/`
- **想了解技术实现** → `04-technical/`
- **想查看开发任务** → `05-tasks/task_mvp.md`

### 按阶段查找

- **新手上手** → `01-project/项目说明.md`
- **功能开发** → `02-features/` + `05-tasks/`
- **工具使用** → `03-tools/`
- **问题排查** → `04-technical/`

## 📝 文档维护

1. **新增文档**：请根据文档类型放入对应分类目录
2. **更新文档**：修改后请更新文档的"最后更新"日期
3. **文档命名**：使用中文描述性名称，清晰易懂
